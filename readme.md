## 安装EMQ
下载地址：https://www.emqx.io/downloads#broker

程序包下载后，可直接解压启动运行，例如 Mac 平台:
```
unzip emqx-macosx-v3.1.0.zip && cd emqx

# 启动emqx
./bin/emqx start

# 检查运行状态
./bin/emqx_ctl status

# 停止emqx
./bin/emqx stop
```

控制台地址: http://127.0.0.1:18083，默认用户名: admin，密码：public
